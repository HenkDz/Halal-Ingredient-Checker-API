# Check a single ingredient
import requests

response = requests.get(
    "https://your-api.com/api/v1/ingredient/gelatin",
    headers={"X-API-Key": "your-api-key"}
)
print(response.json())

# Batch check multiple ingredients
response = requests.post(
    "https://your-api.com/api/v1/batch",
    headers={
        "X-API-Key": "your-api-key",
        "Content-Type": "application/json"
    },
    json={"ingredients": ["gelatin", "whey", "E471"]}
)

# Scan a product barcode
response = requests.get(
    "https://your-api.com/api/v1/barcode/3017620422003",
    headers={"X-API-Key": "your-api-key"}
)

Endpoints

Complete API
reference

Everything you need to build halal verification into your application.

GET /api/v1/ingredient/{name} Look up a single ingredient by name or E-number
POST /api/v1/check Check a list of ingredients at once
GET /api/v1/barcode/{ean} Scan a product via EAN barcode (Open Food Facts)
POST /api/v1/batch Batch check up to 50 ingredients (Pro+)
POST /api/v1/auth/register Register to get a free API key
GET /api/v1/auth/subscription Check your current subscription status
GET /api/v1/health API health check and status
